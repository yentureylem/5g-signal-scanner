### Path Loss & Coverage

The app approximates path loss for an urban macro cell:

- Center frequency around **3.5 GHz** (typical 5G NR band).  
- Distance-dependent path loss inspired by **Okumuraâ€“Hata**:  

\[
PL(d) \approx A(fc, h_t, h_r) + B(h_t)\log_{10}(d)
\]

- Transmit power is fixed; predicted **Rx power** (in dBm) is:

\[
P_{rx} \approx P_{tx} - PL(d)
\]

- Link quality is categorized by Rx power thresholds (e.g. > âˆ’80 dBm = â€œExcellentâ€, < âˆ’100 dBm = â€œPoorâ€).

The 2D coverage heatmap is generated by:

- Building a mesh grid around the base station.
- Computing distance from each grid point to the cell center.
- Applying the same path loss formula to obtain a **power map**.

### Throughput vs SINR

Throughput is computed as a smooth, saturating function of SINR:

- Low SINR â†’ near 0 Mbps.  
- Moderate SINR â†’ rapidly increasing throughput.  
- High SINR â†’ saturation around a target (e.g. 100 Mbps).
